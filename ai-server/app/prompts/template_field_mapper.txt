[1순위: 완전 일치 매핑]
필드명이 정확히 같은 경우 (예: 품목명 → 품목명)
confidence: 1.0

[2순위: 유사어 매핑]
아래 동의어 그룹 기준으로 같은 그룹이면 동일 필드로 처리:
- 품목명 = 제품명 = 항목명 = 공종명 = 작업명 = 품명
- 규격 = 사양 = 스펙 = spec = 형식
- 수량 = qty = 개수 = 갯수
- 단위 = 단가단위 = 계산단위
- 단가 = 견적단가 = 공급단가 = 납품단가 = 단가(원) = 업체단가
- 금액 = 총액 = 합계금액 = 공급금액 = 납품금액 = 공급가액
- 비고 = 특기사항 = 메모 = 참고 = 비고란
- 업체명 = 회사명 = 공급사 = 납품업체 = 거래처
- 순번 = NO = 번호 = 일련번호 = 순서
- 표준단가 = 기준단가 = 시장단가 = 표준시장단가
- 작성일 = 작성일자 = 견적일 = 발행일 = 날짜
- 작성자 = 담당자 = 작성담당 = 기안자
confidence: 0.8

[3순위: 위치 기반 추론]
- 첫 번째 텍스트 컬럼 → 품목명 가능성 높음
- 마지막 숫자 컬럼 → 금액 가능성 높음
- 날짜 형식 값 → 날짜 필드
- 한글 고유명사 + 주식회사/㈜ → 업체명 필드
confidence: 0.5

[4순위: 매핑 불가]
- 1~3순위 모두 해당 없으면 → null로 처리
- 절대 추측하여 채우지 않음
confidence: 0.0

[단일 셀 (single_cell)]
- 하나의 특정 셀에 단일 값 채우기
- 예: 작성자 → M4:N4 셀에 "홍길동" 입력
- 원문에서 해당 값을 찾아 그대로 입력
- 없으면 null (빈 셀 유지)

[반복 행 (repeat_row)]
- 항목 수만큼 행을 아래로 반복하여 채우기
- 시작 행 번호 정확히 지킬 것
- 각 행에 item_name, spec, quantity, unit, unit_price, amount 순으로 채우기
- 원문에 없는 값은 null

[업체 반복 컬럼 (vendor_repeat_column)]
- 업체 수만큼 컬럼 그룹이 오른쪽으로 반복
- 각 업체 그룹에: 단가, 금액 (또는 규격, 수량, 단가, 금액)
- 업체 순서는 원문에 등장한 순서 유지
- 없는 업체 데이터는 null

[메타 정보 (meta_single)]
- 작성일, 작성자, 제목, 회사명 등 단일 값
- 원문 맨 앞 또는 헤더 영역에서 추출
- 날짜는 YYYY-MM-DD 또는 YYYY.MM.DD 형식으로

7. 단위: 원문 단위 그대로 유지
